# mistershell_session_policy_acl (Data Source)

Reads a MisterShell session-policy command ACL. Look up by id or by exact name (must match exactly one). Reads resolve through the list endpoint (there is no GET-single for ACLs).

## Example Usage

```terraform
# Look up a session-policy ACL by its integer ID.
data "mistershell_session_policy_acl" "by_id" {
  id = 42
}

# Look up a session-policy ACL by its exact name.
data "mistershell_session_policy_acl" "by_name" {
  name = "dangerous-commands"
}

output "acl_patterns" {
  value = data.mistershell_session_policy_acl.by_name.patterns
}
```
