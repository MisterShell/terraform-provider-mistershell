# mistershell_ai_tool (Data Source)

Reads a MisterShell AI tool (read-only, backend-builtin). Look up by id or by exact name to obtain the tool id for an mistershell_ai_agent's tool_ids.

AI tools are **backend-builtin and read-only**: MisterShell registers them from
a code-defined registry on startup. They cannot be created, updated, or deleted —
there is no `mistershell_ai_tool` resource. The primary use of this data source is
to **resolve a tool's integer ID** so you can reference it from an
[`mistershell_ai_agent`](../resources/ai_agent.md)'s `tool_ids`.

## Example Usage

```terraform
# AI tools are backend-builtin and read-only. Look one up by its exact name to
# resolve the integer ID you need for an agent's tool_ids.
data "mistershell_ai_tool" "search" {
  name = "search"
}

data "mistershell_ai_tool" "inspect" {
  name = "inspect"
}

# You can also look up a tool directly by ID.
data "mistershell_ai_tool" "by_id" {
  id = 1
}

output "search_tool_permission" {
  value = data.mistershell_ai_tool.search.required_permission
}

# Feed the resolved tool IDs into an agent's tool_ids.
resource "mistershell_ai_agent" "assistant" {
  name = "inventory-assistant"
  type = "chat"

  tool_ids = [
    data.mistershell_ai_tool.search.id,
    data.mistershell_ai_tool.inspect.id,
  ]
}
```

## Schema

### Optional

- `id` (Number) AI tool ID. Use for direct lookup.
- `name` (String) AI tool name. Used as an exact-match search filter when id is not specified.

### Read-Only

- `created_at` (String) Creation timestamp.
- `description` (String) Human-readable description of the tool.
- `handler` (String) Backend handler identifier for the tool.
- `input_schema` (String) The tool's JSON Schema describing its input parameters.
- `required_permission` (String) Permission required to invoke the tool, if any.

## Lookup arguments

Specify exactly one of:

| argument | behavior |
|---|---|
| `id` | Direct lookup by integer ID. |
| `name` | Exact-match lookup by name (fuzzy server search, then exact `name ==` filter). Errors if zero or more than one tool matches. |
